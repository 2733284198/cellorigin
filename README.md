# cellorigin
基于cellnet的服务器框架, 以及配套客户端框架

# 定位
项目的快速启动框架

# 特性

* 基于Protobuf的导表工具及通用数据配置系统

* 配置表索引,关联代码自动生成

* 基于UGUI的UI控件绑定代码自动生成


# 文档请参考
https://github.com/davyxu/cellorigin/tree/master/doc

# 开发环境

* Unity3D 5.x+

* Golang 1.5+
